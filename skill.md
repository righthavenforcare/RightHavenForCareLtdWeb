# Right Haven For Care Website Skill

## Purpose
This file captures the reference website structure, content flow, and design ideas for recreating Right Haven For Care with minor improvements.

## Reference website
URL: https://www.righthavenforcare.co.uk/

### Key sections
1. Header/navigation
   - Logo / brand name
   - Links: Home, About Us, Our Services, Latest News, Contact Us
   - Phone CTA: ☎️ 07368461829

2. Hero/intro section
   - Promotional banner: "Promotional Offer! Care Service starting from just £19 per hour"
   - Headline: "Your Care, Our Commitment"
   - Subheadline: "Compassion Delivered At Home"
   - Short supporting text about local exceptional care services
   - Image or hero illustration

3. Support/benefit section
   - Title: "How we support you"
   - Short message about personalized homecare, independence, confidence, short-term or ongoing care
   - Button CTA: "Check it out →"
   - Supporting image

4. Values section
   - Title: "Our Values"
   - Six value cards: Love, Respect, Compassion, Integrity, Humility, Dignity
   - Each card includes a short statement about that value

5. Call to action section
   - Title: "Join Right Haven For Care?"
   - Emotional supporting paragraph
   - Book Now button linking to Contact page

6. Footer
   - Links: Quick Start, About Us, Privacy Policy, Terms & Conditions
   - Small trust note: "We are registered with CQC"
   - Developer credit: "Designed and Developed By Delta Techs"

## Content and page goals
- Establish trust and warmth for visitors seeking home care.
- Make contact and booking easy with visible CTAs.
- Emphasize values and personal care, not just services.
- Keep the homepage clean, focused, and mobile-friendly.

## Design ideas and improvements
- Use a soft, accessible color palette with warm neutrals and calm accent colors.
- Improve typography with a strong hero font plus readable body text.
- Add distinct CTA buttons in hero and contact sections.
- Strengthen sections with clear cards, spacing, and iconography.
- Add a focused service summary or feature list if desired.
- Consider a small testimonial or trust strip to boost credibility.
- Make navigation sticky or easy to use on mobile.
- Use a responsive layout with consistent spacing and content hierarchy.
- Ensure the site is accessible: good contrast, keyboard-friendly links, alt text.

## Minor improvements to make it better
- Add a visible contact form or contact section on the homepage.
- Replace the generic "Check it out →" link with a section-specific CTA like "View Services".
- Add service highlights or cards under "Our Services" to preview offerings.
- Consider a soft background pattern or gradient in the hero for warmth.
- Improve footer labeling to match actual content (e.g. Services, About, Legal).
- Use real brand imagery or illustrations that reflect care and connection.
- Add smooth scroll behavior for navigation links.

## Notes for implementation
- The current project uses Next.js app router and TypeScript.
- Build the homepage as a composable page in `app/page.tsx` using reusable sections.
- Keep styles in `app/globals.css` and consider simple utility classes.
- Preserve the content hierarchy: hero, support, values, CTA, footer.
- Maintain a mobile-first responsive design.

## Reference for future work
Use this file as a single source of truth while rewriting the homepage and related site pages. Keep improvements small and focused on usability, trust, and conversion.
