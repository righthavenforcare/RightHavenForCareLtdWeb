# Right Haven For Care

A responsive care services website built with Next.js, React, TypeScript, and Tailwind CSS.

## Pages

- Home
- About
- Services
- Latest News
- Contact

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Tech

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
